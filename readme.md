## Live Site
Once deployed to GitHub Pages, view at: [https://yourusername.github.io/marriott-construction](https://yourusername.github.io/marriott-construction)

## Features
- Responsive design (mobile/desktop)
- Custom text-based logo with gold accents
- Hero section with call-to-action
- Services overview
- Embedded payment form via JotForm
- Contact footer

## Setup on GitHub Pages
1. Upload `index.html` to your repo root.
2. Add `.nojekyll` (empty file) to prevent Jekyll processing.
3. Go to **Settings** > **Pages** > Source: "Deploy from branch" > Branch: `main` > Folder: `/ (root)`.
4. Site live in 1-2 minutes!

## Customization
- Edit phone/email in `index.html` footer.
- Replace `[Your City/Area]` in hero.
- Add images to `/assets/` folder and link them.
